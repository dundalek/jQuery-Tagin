jQuery  Tagin

Simple jQuery Tag Input plugin with advanced features.

Features

  • uses jQuery UI themes
  • autocomplete
  • open source (MIT license)
  • advanced features

    autoresize, tag transformations, input character restrictions,
    allow tags only from autocomplete, cursor navigation between tags

  • cross browser

    tested in Chrome, Firefox, Opera and Internet Explorer 6 7 8

Demo

Try the plugin for yourself, type some text and press tab key to create new tag.
Try to type PHP, Python, Java, JavaScript to see auto-completion in action.
Navigate wit left and right arrow key between tags.
Press backspace or delete to remove tags.
